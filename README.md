Student Management System

A mini Student Management System built using **Java** and **Hibernate**.

Project Overview
This project allows you to:
- Add a new student
- View all students
- Update student details
- Delete a student

It demonstrates basic CRUD operations using Hibernate and connecting Java objects to a database.

Tech Stack
- Java
- Hibernate ORM
- PostgreSQL 
- Maven

Concepts Used
- Hibernate `@Entity`, `@Id`
- Session and Transaction management
- CRUD operations (Create, Read, Update, Delete)
- Hibernate configuration (`hibernate.cfg.xml`)

Future Improvements
- Add Courses and mapping with Students (`@OneToMany`, `@ManyToOne`)
- Implement Marks and grading system
- Add search and filtering features
- Convert to Spring Boot application for REST APIs
